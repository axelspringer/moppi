Get a Package

An package object contains a package.

REQUEST

GET

/packages/{package}/{revision}
Parameters
revision int 1

Revision of the package in form of an integer
universe string 1

Name of the universe in form of a string

RESPONSE

Headers
Content-Type application/json
200 application/json
BODY SCHEMA 
                        {
    chronos: [],
    marathon: []
    install: {},
    uninstall: {}
}
